Weather Data API

URL format:- http://127.0.0.1:5000/api/v1/station/date

For one data of one station:- http://127.0.0.1:5000/api/v1/10/1998-10-25

For all data of one station:- http://127.0.0.1:5000/api/v1/10

For all data of one station for a particular year:- http://127.0.0.1:5000/api/v1/yearly/10/1998
